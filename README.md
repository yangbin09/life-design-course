# Life Design Course

This is a React application for the Life Design Course, originally based on a demo.

## Features
- Interactive Life Dashboard
- Odyssey Plans
- Good Time Journal
- Design Thinking Mindsets

## Development
```bash
npm install
npm run dev
```

## Docker Deployment
```bash
docker build -t life-design-course .
docker run -p 80:80 life-design-course
```
